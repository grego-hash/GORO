# GORO 2.3

GORO is a Windows desktop estimating and workbook workflow app built specifically for DFH Construction.

It supports preconstruction and project documentation workflows, including schedule management, CSV-based workbook operations, milestone snapshots, and PDF-to-table import with OCR for scanned files.

## Status

Project Management tools are currently in the developmental phase.

## Built With

- Python
- PyQt
- Installer-based Windows deployment
